# Employee Management System (C)

A console-based Employee Management System developed using C programming language.

##  Project Description
This project allows users to manage employee records through a menu-driven interface. It demonstrates fundamental programming concepts in C.

##  Features
- Add new employee
- Display all employees
- Search employee by ID
- Delete employee record

## Technologies Used
- C Language
- Structures
- Arrays
- Functions
- Loops & Conditional Statements

## Learning Outcome
- Improved understanding of structured programming
- Hands-on practice with data management
- Strengthened problem-solving skills
